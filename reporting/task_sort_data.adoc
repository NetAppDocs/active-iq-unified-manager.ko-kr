= 데이터 정렬
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
결과를 정렬하려면 열을 클릭하고 오름차순 또는 내림차순을 지정합니다. 데이터 정렬은 보고서에 필요한 정보의 우선 순위를 지정합니다.

.단계
. 열 맨 위를 클릭하여 가장 중요한 정보가 보기 맨 위에 표시되도록 결과를 정렬합니다.
. 저장하지 않은 보기의 이름을 지정하여 변경 내용을 저장합니다.
