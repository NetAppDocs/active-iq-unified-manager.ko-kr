= 대시보드에서 클러스터 성능 모니터링
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager 대시보드에서는 이 Unified Manager 인스턴스에서 모니터링하는 모든 클러스터의 높은 성능 상태를 표시하는 몇 가지 패널을 제공합니다. 이를 통해 관리되는 클러스터의 전반적인 성능을 평가하고 식별된 특정 이벤트를 신속하게 확인, 검색 또는 할당할 수 있습니다.
