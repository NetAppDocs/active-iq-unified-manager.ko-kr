= Unified Manager용 7-zip 소프트웨어 패키지를 구성합니다.
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Red Hat Enterprise Linux 시스템에 Unified Manager를 성공적으로 설치하려면 시스템에 7-zip 소프트웨어 패키지에 대한 접근 권한이 설정되어 있어야 합니다. 네트워크 접속이 가능하고 7-zip URL이 방화벽에 의해 차단되지 않은 경우, 7-zip은 설치 또는 업그레이드 과정 중에 자동으로 시스템에 다운로드됩니다. NetApp 자동 다운로드 방식을 권장합니다.

Unified Manager가 7-zip 소프트웨어 패키지를 자동으로 다운로드할 수 있도록 방화벽에서 다음 URL이 차단 해제되어 있는지 확인하십시오.
`https://7-zip.org/a/7z2301-linux-x64.tar.xz`

또는 7-zip을 미리 설치할 수도 있습니다.



== 7-zip 패키지를 미리 설치하세요.

7-zip은 수동으로 다운로드하거나 함께 제공되는 스크립트를 사용하여 미리 설치할 수 있습니다. 다음 두 가지 방법 중 하나를 선택하십시오.



=== 함께 제공되는 스크립트를 사용하여 다운로드하세요.

.단계
. 실행하세요 `install7zip.sh` Unified Manager 압축 파일에 포함된 스크립트입니다.




=== 수동으로 다운로드

7-zip 패키지는 반드시 공식 웹사이트에서만 다운로드하세요.

.단계
. 공식 사이트에서 7-zip 패키지(버전 23.01)를 다운로드하세요.
+
`https://7-zip.org/a/7z2301-linux-x64.tar.xz`

. 해당 파일을 RHEL 가상 머신에 복사하십시오.
+
[source, CLI]
----
7z2301-linux-x64.tar.xz
----
. 압축 파일을 해제하세요:
+
[source, CLI]
----
sudo tar -xf 7z2301-linux-x64.tar.xz
----
. 이동하세요 `7zz` 파일을 `/usr/bin/` 예배 규칙서:
+
[source, CLI]
----
sudo cp 7zz /usr/bin/
----
. 실행 권한을 설정하세요:
+
[source, CLI]
----
sudo chmod 755 /usr/bin/7zz
----




== 설치를 확인하십시오

7-zip 패키지가 시스템에 성공적으로 설치되었는지 확인하십시오.

[source, CLI]
----
7zz -version
----
