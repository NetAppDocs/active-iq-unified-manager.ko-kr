= 사용자 보기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용자 페이지에서 Unified Manager를 사용하여 스토리지 오브젝트 및 데이터를 관리하는 사용자 목록을 볼 수 있습니다. 사용자 이름, 사용자 유형, 전자 메일 주소, 사용자에게 할당된 역할 등 사용자에 대한 세부 정보를 볼 수 있습니다.

* 필요한 것 *

애플리케이션 관리자 역할이 있어야 합니다.

.단계
. 왼쪽 탐색 창에서 * 일반 * > * 사용자 * 를 클릭합니다.
