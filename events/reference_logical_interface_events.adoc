= 네트워크 인터페이스(LIF) 이벤트입니다

[role="lead"]
네트워크 인터페이스 이벤트는 잠재적인 문제를 모니터링할 수 있도록 네트워크 인터페이스(LIF) 상태에 대한 정보를 제공합니다. 이벤트는 영향 영역별로 그룹화되며 이벤트 및 트랩 이름, 영향 수준, 소스 유형 및 심각도를 포함합니다.
