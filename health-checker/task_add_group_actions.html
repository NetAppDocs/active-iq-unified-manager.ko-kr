단계

  1. 왼쪽 탐색 창에서 * 스토리지 관리 * > * 그룹 * 을 클릭합니다.

  2. 그룹 작업 * 탭에서 * 추가 * 를 클릭합니다.

  3. 그룹 작업 추가 * 대화 상자에서 작업의 이름과 설명을 입력합니다.

  4. Group * (그룹 *) 메뉴에서 작업을 구성할 그룹을 선택합니다.
