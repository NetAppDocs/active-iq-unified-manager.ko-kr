= 스토리지 VM 피어 보기
:icons: font
:imagesdir: ../media/


[role="lead"]
스토리지 VM 피어 페이지를 사용하여 기존 스토리지 VM 피어 및 해당 속성을 확인하고 추가 스토리지 VM이 필요한지 여부를 확인할 수 있습니다.

* 필요한 것 *

애플리케이션 관리자 또는 스토리지 관리자 역할이 있어야 합니다.

.단계
. 왼쪽 탐색 창에서 * 보호 * > * 스토리지 VM 피어 * 를 클릭합니다.
+
스토리지 VM 피어 및 해당 속성 목록이 표시됩니다.
